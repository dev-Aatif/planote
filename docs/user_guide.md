### First Run
Upon launching Planote for the first time, you will be greeted by the main dashboard. The interface is divided into:
-   **Sidebar**: Navigation between Inbox, Today, Upcoming, Projects, and Labels.
-   **Main View**: Displays your tasks or notes for the selected category.

## ðŸ“ Using Notes

Planote features a simplified, distraction-free note editor designed to help you focus on your thoughts.

### Creating a Note
1.  Navigate to the **Notes** section (or a specific Project).
2.  Click the **+ (New Note)** button.
3.  Enter a **Title** for your note.
4.  Start typing your content in the main editor area.

### Using Headings
To structure your notes, Planote supports **Headings**.
-   Use the dropdown menu in the toolbar to apply Heading styles (Normal, H1, H2, H3, etc.).
    -   Use **H1** for main sections.
    -   Use **H2** and **H3** for subsections.
-   **Normal Text**: Select "Normal" to revert to standard paragraph text.

### Searching Notes
Use the search bar at the top of the application to quickly find notes by title or content.

### Adding a Task
1.  Click the **New Task** button or simply start typing in the "Add a task..." field.
2.  Press **Enter** to save the task.

### Organizing Tasks
-   **Projects**: Group related tasks into Projects.
-   **Labels**: Tag tasks across different projects for easy filtering.
-   **Due Dates**: Assign dates to keep track of deadlines.
